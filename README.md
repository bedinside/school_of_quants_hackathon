# school_of_quants_hackathon
School of Quants Hackathon 2025

1) Участникам необходимо обучить модель на основе фичей из X_train и лейблов (правильных ответов) из y_train 2) На основе X_test участники делают предсказание значений y_test (скрыт от участников) с помощью обученных моделей 3) Полученные и загруженные предсказания участников сравниваются с настоящим файлом y_test, высчитывается метрика качества

Датасет: в приложении (X_test.csv, X_train.csv, y_train.csv)

Фичи: – credit_number_for_user - Порядковый номер кредитного продукта в кредитной истории. Большему номеру соответствует продукт с более поздней датой открытия – days_since_confirmed - Дней с даты подтверждения информации по кредиту до даты сбора данных* – maturity_plan - Плановое количество дней с даты открытия кредита до даты закрытия* – maturity_fact - Фактическое количество дней с даты открытия кредита до даты закрытия* – credit_limit - Кредитный лимит* – next_payment_sum - Сумма следующего платежа по кредиту* – sum_left_to_pay - Оставшаяся невыплаченная сумма кредита* – current_overdue_debt - Текущая просроченная задолженность* – max_overdue_debt - Максимальная просроченная задолженность* – full_credit_cost - Полная стоимость кредита* – overdues_Xd_Yd - Число просрочек сроком между X дней и Y дней (либо менее 5 дней/более 90 дней)* – no_overdues_Xd_Yd - нет просрочек сроком между X дней и Y дней (либо менее 5 дней/более 90 дней) – enc_paym_{0..N} - Статусы ежемесячных платежей за последние N месяцев* – credit_type - Тип кредита* – credit_currency - Валюта кредита***

Лейбл: flag - наступил ли дефолт

Примечание:

- область значений поля разбивается на N непересекающихся промежутков, каждому промежутку случайным образом ставится в соответствие уникальный номер от 0 до N-1, значение поля заменяется номером промежутка, которому оно принадлежит
** - флаг (binary variable) *** - каждому уникальному значению поля случайным образом ставится в соответствие уникальный номер от 0 до K, значение поля заменяется номером этого значения
